# SANBER-45_UI_TESTING

How to Start:

Install:
  1. Node JS
  2. Cypress.io
  3. Install package.json with command

npm install
npm install cypress --save-dev

How to run

  1. Connect to your device android or IOS
  2. Setup desired capabilities in file wdio.conf.js
  3. Run test

npx cypress open
npx cypress run
